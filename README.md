# learn-python-by-doing
An interactive Python tutorial with modular theory, excercises, solutions and automated tests. Clone, install, code, test!

Тут в будущем будет больше всего

## Для начала работы
В консоли
```
git clone https://github.com/MaLeDaNi/learn-python-by-doing
```
Появится папка с проектом.

Открываем её в любом IDE.

(В будущем будет ещё вариант с архивом, но пока только так)

### Для активации venv (В корне проекта learn-python-by-doing)
В консоли
```
py -m venv venv
venv\Scripts\activate
```

## Работа

Вся работа осуществляется в соответствующем модуле в папке **Tasks**.
Весь код пишется в там.

Теория к каждому заданию находится в папке **Theory** и название соответствует файлу с заданием из папки **Tasks**.

Если вы не знаете как решить задание, то есть папка **Solutions**, где лежат решения каждой задачи(списывание оттуда на вашей совести. Только в случае крайней необходимости)

## Тестирование

В консоли
```
pip install pytest
```

Для тестирования задания нужно перейти в соответсвующую папку с тестами.
У каждого модуля она своя

Допустим, модуль 01_syntax_basics.

Переходим в папку с тестами для этого модуля
```commandline
cd 01_syntax_basics/Tests/
```

Запускаем нужный тест(Например тест на задание с комментариями)
```commandline
pytest 01_test_comments.py -s
```

Переходим обратно в корень проекта(если требуется)
```commandline
cd ../../
```
